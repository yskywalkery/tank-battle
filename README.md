# 坦克大战 - Tank Battle

经典坦克大战 HTML 游戏，在浏览器中直接运行，无需安装任何依赖。

## 快速开始

用浏览器打开 `index.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` 或 `↑` `↓` `←` `→` | 移动坦克 |
| `空格` | 射击 |
| `R` | 游戏结束后重新开始 |

## 游戏规则

### 基本玩法

- 控制绿色坦克，消灭场上所有敌人即可进入下一波
- 每波击杀目标：**4 + 波次 × 2** 个敌人
- 初始拥有 **3 条生命**，被击中后短暂无敌并回到出生点
- 生命耗尽则游戏结束

### 地图元素

| 元素 | 说明 |
|------|------|
| 🧱 砖墙 | 可被子弹摧毁 |
| 🛡 钢墙 | 不可摧毁，永久障碍物 |
| ⚡ 加速 | 8 秒内移动速度提升 |
| 🔥 速射 | 8 秒内射击冷却大幅缩短 |
| 🛡 护盾 | 临时无敌状态 |
| 💣 炸弹 | 立即消灭场上所有敌人 |

### 敌人

- 普通敌人（红色）：1 HP，一枪击杀
- 精英敌人（橙色）：2 HP，需两枪击杀，从第 3 波开始出现
- 敌人会追踪玩家并在对齐时射击
- 击杀敌人有 15% 概率掉落道具

### 波次

- 共 3 种地图循环（简单 → 堡垒 → 迷宫）
- 波次越高，敌人越多、生成越快、AI 射击更频繁

## 技术说明

纯前端实现，使用 HTML5 Canvas 渲染，无任何外部依赖。
